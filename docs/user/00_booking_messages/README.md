# Booking Messages — Overview

For booking message behavior, read Booking Rules first: booking_rules/README.md, booking_rules/actions.md, and booking_rules/rule-types.md.
Booking message delivery, reminders, and notification automation are configured in Booking Rules, not in Actions After Booking.

## Click-by-click: Configure a booking message

1. Open the rule editor: [/mod/booking/edit_rules.php?contextid=1](/mod/booking/edit_rules.php?contextid=1).
2. Click Add rule.
3. Select a template from [Booking Rules Templates](../booking_rules/templates.md) or choose trigger, condition, and action manually.
4. In the action section, choose a mail action and write subject/body.
5. Save the rule and set it active.
6. Trigger a test booking and verify message delivery.
