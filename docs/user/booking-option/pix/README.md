# pix — Screenshots and Images

This folder contains screenshots and images used in the booking option documentation.

## Naming convention

```
<page-slug>-<description>.png
```

Examples:
- `general-invisible.png` — screenshot of the "Invisible" checkbox in the General section
- `dates-recurring.png` — screenshot of the recurring date form
- `availability-conditions.png` — overview of the availability conditions panel

## Adding screenshots

Screenshots should be taken from a standard Moodle installation with the booking plugin installed.
Preferred dimensions: **1200 × 800 px** or wider, PNG format.
